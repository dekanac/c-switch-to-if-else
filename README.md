# c-switch-to-if-else
Takes C code as a input, transform switch statements to if/else.

**- Installation of Clang:**
    
    sudo apt-get install clang
    sudo apt-get install bison
    sudo apt-get install flex
    
**- Instalation:**

    make
    
    REMOVE:
    make clean
    
**- Usage:**

    ./swi2else < FILE
